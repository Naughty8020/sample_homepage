## development rules

### ・各ファイルの行数は88行以内
### ・UIとロジックのファイルは切り出す
### ・型を定義する
### ・関数は名前付きエクスポートで書く
### ・命名規則は他のファイルと型式を揃える
### ・コミットメッセージは[操作名(場所):やったこと]
### ・差分が大きくならないようにコミットする

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
